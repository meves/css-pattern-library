Создание библиотеки компонентов KSS:
потребуется nodejs

Подготовительная часть:
создаем каталог проекта
переходим в него и выполняем npm init -y (инициализация проекта)
установка kss: npm install --save-dev kss
создается конфигурационный файл kss: kss-config.json
	{
		"title": "Му pattern library"
		"source": [
			"./css"					- каталог файлов css
		L
		"destination": "docs/",		- каталог назначения
		"css": [
			"../css/styles.css		- исходные файлы стилей (возможно пропроцессоров)
		L
		"js": [
			”../js/docs.json		- каталог сценариев
		]
	}

добавить в package.json команды выолнения:
"scripts": {
	"build": "kss --config kss-config.json",
	"test": "echo V'Error: no test specified\" && exit 1"
},	
запуск: npm run build

Создание компонента:
	пример в любом файле SASS


Теперь выполните команду npm run build в оболочке командной строки. KSS
создаст каталог docs, содержащий файл section-media.html. Откройте эту страницу
в браузере, чтобы увидеть библиотеку компонентов.

Можно создавать вариации модуля, примеры там же

В каталоге css создайте файл homepage.md.
